# AI Model Icons

Place the SVG icon files for each AI model in this directory with the following names:

- `chatgpt.svg` - ChatGPT icon
- `claude.svg` - Claude icon
- `gemini.svg` - Gemini icon
- `mistral.svg` - Mistral icon
- `deepseek.svg` - DeepSeek icon
- `llama.svg` - LLaMA icon
- `grok.svg` - Grok icon
- `perplexity.svg` - Perplexity icon
- `default.svg` - Default fallback icon (already included)

Icons should be:
- SVG format
- Approximately 32x32px or scalable
- Optimized for dark theme backgrounds
- Light/white colored for visibility on dark backgrounds

You can find official logos at:
- https://opttab.com/ai-model-logos
- https://lobehub.com/en/icons/
- Or use the official brand assets from each company
